发表于 2021-05-15 分类于 系统编程

文件与I/O函数

open函数

  • 头文件：<sys/types.h>, <sys/stat.h>, <fcntl.h>
  • 返回值：成功返回非零整数，失败返回-1。
  • 执行类型：O_RDONLY, O_WRONLY, O_RDWR, O_CREAT

close函数

  • 头文件：<unistd.h>
  • 函数原型： int close(int fd);
  • 参数: fd 文件描述符
  • 返回值：成功返回0， 失败返回-1。

read函数

  • 头文件：<unistd.h>
  • 函数原型 int read(int fd, void *buf, int size);
  • 返回值： 失败返回 -1； 文件读完返回0； 读取的字节数返回的数大于0；

write函数

  • 头文件<unistd.h>
  • 函数原型： int write(int fd, void *buf, int size);
  • 返回值：写入的字节数, 失败返回-1。

lseek函数

  • 头文件<sys/types.h>, <unistd.h>
  • 功能：修改文件偏移量（读写位置）。
  • 函数原型： off_t lseek(int fd, off_t offset, int whence)
  • 参数： fd：文件描述符 对offset的解释与whence的值是相关的： 当whence是SEEK_SET, 则将该文件的偏移量设置为距文件开始处offset个字节 当whence是SEEK_CUR,则将该文件的偏移量设置为其当前值加上offset，offset可正可负数。 当whence是SEEK_END,则将该文件的偏移量设置为文件长度加上offset，offset可正可负数。
  • 返回值：成功返回较文件起始位置向后的偏移量，失败返回-1.
